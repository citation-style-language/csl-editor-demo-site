To use in your reference manager


Mendeley

Mendeley has its own instance of this CSL Editor hosted at http://csl.mendeley.com. If you use that CSL Editor instance Mendeley will host your style, install it to your machine and help to share the style with other CSL users.

If you want install the current edited style in your Mendeley Desktop 1.7 or above follow these steps:

  • Click the "Save to disk" button above to save your style
  • In Mendeley Desktop, go to the View menu, Citation Style, More Styles...
  • Drag and drop the saved style into the style list

Zotero

Use the above button to save your .csl file wherever you like, then:

  • Firefox plugin

    Drag the saved file into any open Firefox window.

  • Standalone

    In Zotero standalone go to Preferences->Cite tab->Styles, and click the '+' button to add your new style.

Papers 2

Use the above button to save your .csl file to the 'Styles' subfolder within your Papers2 library folder. More info.

Docear4Word

Use the "Save To Disk" button above to save this citation style in your Docear4Word style folder. The style folder is located at

c:\ProgramData\Docear4Word\Styles\

If the folder c:\ProgramData\ is not visible, you need to activate Window's option to show invisible files. To do so, open the Windows Explorer, press ALT, select Tools -> Folder Options -> View -> Hidden files and folders and activate "Show hidden files, folders, and drives".
